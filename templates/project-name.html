Autonomous Rover with SLAM

A ROS-based differential drive rover capable of real-time mapping and localization in an unknown environment.

Project Overview

This project involved the design, construction, and programming of an autonomous mobile robot. The primary goal was to implement Simultaneous Localization and Mapping (SLAM) using a 2D LiDAR sensor and the Robot Operating System (ROS). The rover can explore an unknown indoor environment, build a map, and localize itself within that map in real-time.

  • **Duration:** 12 weeks
  • **Team Size:** 3 (My Role: Lead Robotics Software, Mechanical Integration)
  • **Technologies:** ROS (Noetic), Python, C++, RPLiDAR, Raspberry Pi 4, Arduino, SolidWorks

Mechanical Design & Fabrication

The rover chassis was designed in **SolidWorks** to accommodate all electronic components, including the Raspberry Pi, motor drivers, battery, and LiDAR sensor. Emphasis was placed on modularity and ease of assembly. Components were primarily 3D-printed using PLA.

A key challenge was designing a robust differential drive system with sufficient torque for various floor surfaces, while also ensuring precise encoder feedback for odometry calculations.

Software Implementation (SLAM & Navigation)

The software stack was built entirely on **ROS Noetic**, leveraging existing packages for LiDAR data processing (RPLiDAR driver) and motor control (custom Arduino node).
